Verzeichnis doc/etc:


Mit Hilfe der in diesem Verzeichnis enthaltenen UDO-Quelltexte
und Skripte können folgende Dateien erzeugt werden:

* copying.u -> COPYING.txt, wahlweise in Deutsch oder Englisch.
  Bestandteil der UDO-Distribution für alle Plattformen.

* file_id.u -> FILE_ID.DIZ, wahlweise in Deutsch oder Englisch.
  Bestandteil der UDO-Distribution für bestimmte Plattformen.
  Achtung: plattformspezifische Texte werden in Abhängigkeit von
  Parametern erzeugt, die dem Skript mittels "-D" zu übergeben sind.

* install.u -> INSTALL.txt, wahlweise in Deutsch oder Englisch.
  Bestandteil der UDO-Distribution für alle Plattformen.

* readme.u -> README.txt (Englisch) oder LIESMICH.txt (Deutsch).
  Bestandteil der UDO-Distribution für alle Plattformen.

Bei der Erzeugung dieser Dateien wird auf include-Dateien aus dem
Verzeichnis doc/manual/source zurückgegriffen.

Directory doc/etc:


Using the UDO sources and scripts which are located here, you
can produce the following files:

* copying.u -> COPYING.txt, in german or english.
  Part of the UDO distribution for all platforms.

* file_id.u -> FILE_ID.DIZ, in german or english.
  Part of the UDO distribution for specific platforms.
  Note: platform-specific text parts will be produced depending on
  some parameters, which you must specify with "-D".

* install.u -> INSTALL.txt, in german or english.
  Part of the UDO distribution for all platforms.

* readme.u -> README.txt (english) or LIESMICH.txt (german).
  Part of the UDO distribution for all platforms.

While creating these files, some include files will be loaded from
directory doc/manual/source.
